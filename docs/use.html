Использую

Подборка приятного и удобного софта под Мак, которым пользуюсь каждый день.

Продуктивность

Spark — клевый почтовый клиент.

Things 3 — таск менеджер для GTD.

pinboard.in — онлайн сервис для организации закладок. Мои закладки.

Simplepin — pinboard.in клиент для iOS (несколько лет без апдейтов).

Better Notes — Приятная и минималистичная записная книжка в виде ленты. Очень удобно быстро что-то записать.

Fantastical — Если нужен Google Calendar, то лучше нет ничего.

Разное

Firefox — Очень приятный стал, и можно в приватном режиме все время быть.

Remembear — Тулза для менеджмента уникальными паролями.

Castro — Подкасты, и есть возможность загрузить звук от видео с Youtube.

Day One — Пишу в него всякое личное про себя и семью.

Dropbox — Тут без открытий, уже давно плачу за терабайт и альтернативы не вижу.

Loyalty Cards — Очень минималистичное и тем удобное приложение для хранения всяких скидочных карт со штрихкодами.

PS Deals — Когда ты не упоротый геймер и можешь подождать распродажи. Добавляешь приложение в лист ожидания и приложение само сообщит, когда скидон и пора покупать.

Numi — Калькулятор для Мака, типа скретч пада, в котором можно всякое разное считать, заводить переменные и тп.

Reeder — Пользуюсь только на маке для чтения RSS (через The Old Reader)

Bumpr — Удобная маленькая штука для Мака, которая позволяет выбрать в каком браузере открыть ссылку.

Rectangle — Двигать окошки туда-сюда по экрану.

Alacritty — Клевый и быстрый терминал.

Readwise — Синкаю заметки из Киндла и потом просматриваю.

Instapaper — Сервис для чтения статей, который использую только для того, чтобы раз в неделю посылать интересные статьи на Кидл.

MonitorControl — Маленькая утилитка, которая позволяет регулировать яркость внешнего монитора стандартными кнопками маковской клавы.
