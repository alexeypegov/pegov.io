Использую

Подборка приятного софта и сервисов, которыми пользуюсь каждый или почти каждый день.

Продуктивность

  • Spark. Клевый почтовый клиент, который есть и под iOS и в котором можно отклыдывать сообщения "на потом".
  • Things 3. Таск менеджер для GTD.
  • Itsycal. Календарь для маковского менюбара, в котором удобно открывать созвоны в Zoom.
  • Better Notes. Приятная и минималистичная записная книжка в виде ленты. Очень удобно быстро что-то записать.
  • Up Next. Крохотное приложение, показываюещее следующую встречу в маковском менюбаре.
  • LICEcap. Очень простая программа для записи скринкастов в анимированные гифки, есть и под винду.
  • Bumpr. Удобная маленькая штука для Мака, которая позволяет выбрать в каком браузере открыть ссылку.

Разное

  • Firefox. Всегда сижу в incognito mode.
  • Remembear. Несколько лет назад перестал использовать один пароль для всего.
  • pinboard.in. Очень простой онлайн сервис для организации закладок. Мои закладки.
  • Castro. Слушаю подкасты, и иногда сохраняю интервью с Youtube.
  • Zen Journal. Стараюсь вести дневник каждый день, подмечать интересное вокруг.
  • Dropbox. Уже давно плачу за терабайт и альтернативы не вижу.
  • Numi. Калькулятор, в котором можно всякое разное считать, заводить переменные и тп.
  • Rectangle. Двигать окошки туда-сюда по экрану.
  • Sublime Text. Легкий программерский (и не только) редактор.
  • Reeder. Пользуюсь для чтения RSS (вместе с The Old Reader)
  • Alacritty. Клевый и быстрый терминал.
  • Instapaper. Сервис для чтения статей, который использую только для того, чтобы раз в неделю посылать интересные статьи на Кидл.
  • MonitorControl. Маленькая утилитка, которая позволяет регулировать яркость внешнего монитора стандартными кнопками маковской клавы.
