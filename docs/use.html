Ложное движение

О сайте Фотографирую ВОЙНА

Подборка софта (в основном mac/ios) и сервисов, которыми пользуюсь каждый или почти каждый день.

Продуктивность

  • DoNext для всех дел, по методике GTD (до этого очень долго был Things 3, но нашлось что-то попроще).
  • MeetingBar. Крохотное приложение, показываюещее следующую встречу в верхнем меню, с возможностью добавить шорткат для запуска Зума.
  • Fastmail. Слез с гмейла для личной почты.
  • Alacritty. Замена Terminal.

Разное

  • Firefox. Всегда сижу в incognito mode.
  • Bitwarden. Менеджер паролей. В бесплатной версии есть ограничения (платная стоит $10 в год).
  • pinboard.in. Очень простой онлайн сервис для организации закладок. Мои закладки.
  • Overcast. Для подкастов.
  • Numi. Калькулятор, в котором можно всякое разное считать, заводить переменные и тп.
  • Rectangle. Двигать окошки туда-сюда по экрану.
  • Sublime Text. Легкий программерский (и не только) редактор. Со шрифтом MonoLisa и темой Monokai Pro вообще бомба.
  • ReadKit. Пользуюсь для чтения RSS (вместе с BazQux)
  • Spotify. Тут все понятно)
Changelog

[Febriary, 2, 2023]

Added:

  • ReadKit. Пользуюсь для чтения RSS (вместе с BazQux)

Removed:

  • Itsycal. Календарик для верхнего меню (menubar).

  • Bumpr. Удобная маленькая штука для работы с несколькими браузерами, в которой можно определить какие сайты в каком браузере открывать.
  • Reeder. Пользуюсь для чтения RSS (вместе с The Old Reader)
