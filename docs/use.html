О блоге Использую Фотографирую

Использую

Подборка приятного маковского софта и сервисов, которыми пользуюсь каждый или почти каждый день.

Продуктивность

  • Things 3. Записываю дела, в соответствии с методикой GTD.
  • Itsycal. Календарь, в котором удобно открывать созвоны в Zoom.
  • Up Next. Крохотное приложение, показываюещее следующую встречу в менюбаре.
  • LICEcap. Очень простая программа для записи скринкастов в анимированные гифки, есть и под винду (пока не работает с macOS Big Sur).
  • Bumpr. Удобная маленькая штука, которая позволяет выбрать в каком браузере открыть ссылку.
  • Fastmail. Слез с гмейла для личной почты, доволен.
  • WorkFlowy. Для людей, у которых из всего можно сделать список.
  • Drafts. Для быстрой записи заметок. Записываю содержание встреч, какие-то мысли и тп, а потом обрабатываю и распихиваю в WorkFlowy, в рабочую Wiki или куда-то еще.

Разное

  • Firefox. Всегда сижу в incognito mode.
  • Bitwarden. Менеджер паролей. В бесплатной версии есть ограничения (платная стоит $10 в год).
  • pinboard.in. Очень простой онлайн сервис для организации закладок. Мои закладки.
  • Castro. Слушаю подкасты, и иногда сохраняю интервью с Youtube.
  • Day One. Какое-то время назад стал вести ежедневный дневник, потом решил фотографировать и записывать птиц, которых встречаю, фотографии, которые делаю и тп. В общем — дневник на все случаи жизни.
  • Dropbox. Уже давно плачу за терабайт. Думаю куда-нибудь перелезть, но пока не перелез.
  • Numi. Калькулятор, в котором можно всякое разное считать, заводить переменные и тп.
  • Rectangle. Двигать окошки туда-сюда по экрану.
  • Sublime Text. Легкий программерский (и не только) редактор.
  • Reeder. Пользуюсь для чтения RSS (вместе с The Old Reader)
  • Alacritty. Клевый и быстрый терминал.
  • Instapaper. Сервис для чтения статей, который использую только для того, чтобы раз в неделю посылать интересные статьи на Кидл.
  • MonitorControl. Маленькая утилитка, которая позволяет регулировать яркость внешнего монитора стандартными кнопками маковской клавы.
